# asir_zmq

ZMQ Transport for ASIR

## Installation

Add this line to your application's Gemfile:

    gem 'asir_zmq'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install asir_zmq

## Dependencies

* gem install zmq
** zmq 2.1.4 needs a zmq 2.x library.
** On OS X Macports: sudo port install zmq22.
